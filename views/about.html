About the User Management System

The User Management System is a web application designed to manage users effectively and securely. It provides a streamlined experience for handling user registrations, logins, and data management on the platform.

Features:

  • User Registration: Register new users with secure password encryption using bcrypt.
  • User Login: Secure login system with session handling and validation.
  • Dashboard: Accessible only to logged-in users, providing an overview of user data.
  • Data Storage: MongoDB is used as the database for storing user details.
  • File Uploads: Users can upload their profile pictures securely using multer.
  • Responsive Design: User-friendly interface optimized for various screen sizes.

Technologies used in this project include HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and more.
